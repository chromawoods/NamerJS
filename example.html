Fragment data used in this example:

var fragments = [
  { text: 'first',  pos: 1 },
  { text: 'middle', pos: 2},
  { text: 'last',  pos: 3 },
  { text: 'any',  pos: 0 },
  { text: 'explicit', pos: [1, 3] }
];
text:
Whatever name-fragment you want, for example Donald or Duck
pos:
Tells namer at what position this name-fragment can appear. Use a single integer if it can only appear in one position. 1 means it can only appear as the first part of the name, etc. 0 means that it can appear in ALL positions. Use an array of integers to define any positions you want. In this example, [1, 3] means that the fragment can only appear in the first and third position.
